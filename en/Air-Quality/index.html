About the Project

The Air Quality Monitoring System is an IoT-based solution that tracks and analyzes air pollution parameters in real-time. This project combines hardware sensors with a web-based dashboard to provide comprehensive environmental monitoring capabilities.

Real-Time Monitoring

  • Alert system for hazardous pollution levels

Web Dashboard

System Architecture

  1. Hardware Layer: Sensors connected to microcontroller (Arduino/ESP32)
  2. Data Collection: Sensor readings collected at regular intervals
  3. Communication: Data transmitted via Wi-Fi/MQTT to server
  4. Backend Processing: Python server processes and stores data
  5. Database: Time-series data stored in database
  6. Web Dashboard: Real-time visualization and analysis interface

How It Works

  1. Air quality sensors continuously measure environmental parameters
  2. Microcontroller collects sensor data and packages it
  3. Data is transmitted to the server via Wi-Fi/MQTT protocol
  4. Python backend processes the data and updates the database
  5. Web dashboard fetches and displays real-time data with visualizations
  6. System triggers alerts when pollution levels exceed safe thresholds

Measured Parameters

  • PM2.5 & PM10: Particulate matter concentration
  • CO2 Levels: Carbon dioxide concentration
  • Temperature: Ambient temperature readings
  • Humidity: Relative humidity percentage
  • Air Quality Index (AQI): Calculated overall air quality score

About the Project

This Python application provides a command-line interface for monitoring air quality in cities worldwide. It integrates with OpenWeatherMap's APIs to fetch real-time air quality data, including AQI (Air Quality Index) and pollutant levels.

The system features secure user authentication with bcrypt password hashing, password recovery through security questions, and stores user data in CSV format for simplicity.

Key Features

  • User Registration & Authentication: Secure account creation with email validation and bcrypt password hashing
  • Password Recovery: Security question-based password reset functionality
  • City-Based AQI Retrieval: Fetch air quality data for any city using OpenWeatherMap's Geocoding and Air Pollution APIs
  • Detailed Pollutant Information: Displays AQI index along with individual pollutant levels (CO, NO, NO2, O3, SO2, PM2.5, PM10, NH3)
  • Login Attempt Limiting: Security measure with maximum 5 login attempts
  • CSV Data Storage: Simple file-based storage for user credentials

How It Works

  1. Users register with email, password, and a security question
  2. Passwords are hashed using bcrypt before being stored in the CSV file
  3. After login, users can enter any city name to retrieve air quality data
  4. The application uses OpenWeatherMap's Geocoding API to convert city names to coordinates
  5. Air Pollution API is then called with these coordinates to fetch current AQI and pollutant data
  6. Results are displayed in an easy-to-read format showing overall AQI and individual pollutant levels

Air Quality Data Displayed

  • AQI Index: Overall air quality rating (1-5 scale)
  • CO: Carbon Monoxide concentration
  • NO & NO2: Nitrogen Oxide levels
  • O3: Ozone concentration
  • SO2: Sulphur Dioxide levels
  • PM2.5 & PM10: Particulate matter measurements
  • NH3: Ammonia concentration

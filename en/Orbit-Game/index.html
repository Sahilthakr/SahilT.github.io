Orbit Game

A simple, interactive game where you control an orbiting object around a central body. Built with Unity Engine, this game demonstrates basic orbital mechanics and gravity physics in an engaging way.

About the Game

Orbit Game is an engaging physics-based game that challenges players to control an orbiting object. The goal is to maintain a stable orbit while preventing the object from being pulled into the central body or flying off into space. It's a fun and educational demonstration of orbital mechanics and gravitational forces.

Features

  • Simple Controls: Easy to pick up and play with keyboard or mouse controls
  • Orbital Mechanics: Experience realistic gravity and orbital physics
  • Score Tracking: Compete with yourself to achieve higher scores
  • Progressive Difficulty: The game becomes more challenging as you advance
  • Physics Simulation: Real-time gravitational force calculations

How to Play

  1. Download and run the game executable for your platform
  2. Use keyboard controls to adjust the orbit velocity
  3. Maintain a stable orbit without crashing into the center
  4. Avoid flying off into space by managing your speed
  5. Try to achieve the highest score possible!
